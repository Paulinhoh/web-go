# Configuração de aplicação Go

## Importância

- Papel fundamental na adaptabilidade, portabilidade, testabilidade e flexibilidade da aplicação (ities)
- Permite ajustar o comportamento e até mesmo aparência da aplicação sem precisar alterar o código

Podemos citar algumas situações em que isso é importante

1 - adaptabilidade a ambientes variados
2 - personalização de funcionalidades ou de aparência
3 - facilidades na manutenção e escalabilidade
4 - segurança e gerenciamento de dados sensiveis
5 - melhorias a testabilidade de aplicação

## Implementação

Existem varias maneiras de permitir a configuração de uma aplicação

- parâmetros de linha de comando
- variaveis de ambiente
- arquivos externos (json, csv, toml, properties)
- API de configuração externa
- Banco de dados
- ou até mesmo uma mistura de várias opções
